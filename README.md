# hubot-hoge

A hubot script that does the things

See [`src/hoge.coffee`](src/hoge.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-hoge --save`

Then add **hubot-hoge** to your `external-scripts.json`:

```json
[
  "hubot-hoge"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
